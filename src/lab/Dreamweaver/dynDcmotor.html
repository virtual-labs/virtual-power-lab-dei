SIMULATION GRAPH

This program draws the graph of a dc motor speed, field current and armature current with respect to different values of external armature resistance Ra_ext. The values of Ra
can be adjusted in the range 0-20 ohm using the sliders at the bottom of the
graph and the variation in motor speed and armature current can be studied.

If graph is unable to see, then click here to install missing Java plugin.
